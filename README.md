== TF2Toolbox -- backpack parsing utilities for Team Fortress 2

=== Quickstart

Fill out a 'config.py' file - the 'config.py.example' file will guide you.

Get the requirements.
    pip install -r requirements.txt

Start the server!
    python tf2toolbox.py
